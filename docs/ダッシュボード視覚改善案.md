# ダッシュボード視覚改善案

## 現状の問題点

### 1. **視覚的階層の不明確さ**
- セクションタイトルが小さすぎる（`text-xs`）
- カードタイトルとセクションタイトルの区別がつきにくい
- フォントサイズの階層が不明確

### 2. **カードタイトルの不統一**
- 一部のカード: `text-base`
- 一部のカード: デフォルト（`text-2xl`）
- アイコンのサイズも不統一（`h-5 w-5` vs デフォルト）

### 3. **色の使い方**
- トレンド表示の色が直感的でない（増加=赤、減少=緑は一般的だが、アラートの文脈では逆の方が良いかも）
- カードの背景色が統一されていない

### 4. **余白の問題**
- セクション間の余白（`space-y-8`）が大きすぎる可能性
- カード内の余白が不統一

### 5. **レスポンシブ対応**
- `xl`ブレークポイント（1280px）が大きすぎる
- モバイル表示での見やすさが不十分

### 6. **情報密度**
- カード内の情報が詰まりすぎている箇所と、余白が多すぎる箇所が混在

## 改善案

### 1. **視覚的階層の明確化**
- セクションタイトル: `text-sm` に変更（`text-xs`から）
- カードタイトル: `text-lg` に統一（`text-base`から）
- 説明文: `text-sm` に統一

### 2. **カードタイトルの統一**
- すべてのカードタイトルを `text-lg` に統一
- アイコンサイズを `h-5 w-5` に統一

### 3. **色の改善**
- トレンド表示: 増加=緑、減少=赤に変更（アラート数が減るのは良いこと）
- カードの背景色を統一（白背景）

### 4. **余白の最適化**
- セクション間: `space-y-6` に変更（`space-y-8`から）
- カード内の余白を統一

### 5. **レスポンシブ対応**
- `xl:grid-cols-3` → `lg:grid-cols-3` に変更
- モバイル表示の改善

### 6. **情報密度の調整**
- カード内の余白を適切に調整
- グラフの高さを最適化
